\begin{frame}{Висновки}
\framesubtitle{}
{\scriptsize
У роботі було розглянуто
\begin{itemize}
 \itemsep=1.0mm
 \item Можливі форми організації процесу розробки ігор та місце сценариста у цьому процесі. 
 \item Можливості мультиагентних систем у рамках симуляції ігрових середовищ.
 \item Можливість інтеграції системи для розробки ігрових сценаріїв у процес розробки ігор.
\end{itemize}
Було запропоновано
\begin{itemize}
 \itemsep=1.0mm
 \item Абстрактну архітектуру фреймворку для проектування ігрових сценаріїв.
 \item Оптимальний фреймворк для створення даної мультиагентної системи.
\end{itemize}
У роботі не представлено жодної практичної реалізації спроектованої системи, але в майбутньому планується реалізація фрейморку та створення конкретної системи для проектування ігрових сценаріїв.
}
\end{frame}
